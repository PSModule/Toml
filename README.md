# Toml

Toml is intended to be a PowerShell module for converting TOML strings to PowerShell objects and PowerShell objects to TOML.

## Status

This repository is currently a placeholder/in-progress. The current `ConvertFrom-Toml` command is a stub and throws `NotImplementedException`, so there are no supported TOML conversion commands or usage examples to document yet.

## Contributing

Issues and pull requests are welcome when implementation work begins.
